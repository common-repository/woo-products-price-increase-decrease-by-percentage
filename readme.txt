== Installation ==

How to install:

1. Search for "WooCommerce Products Price Increase / Decrease by Percentage" via Plugins/Add New on your WordPress admin panel. OR Upload woocommerce-ppidbp folder into `/wp-content/plugins/`
2. Activate Plugin
3. Go to submenu: Products > Increase / Decrease prices by percentage
5. Start
